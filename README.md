### css Mocules

-   css classname 중복을 완벽하게 방지할 수 있음.
-   .module.css (ex)Box.module.css
-   import styles from "./Box.module.css" (고유화됨)
-   유용한 상황 : 레거시 프로젝트에 리액트를 도입할 때 (기존에 만든 프로젝트에 리액트 도입)
-   유용한 상황 : CSS 클래스 네이밍 규칙 만들기 귀찮을 때
